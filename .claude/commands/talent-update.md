# Talent-Augmenting Layer — Profile Update

You are running a profile update for the Talent-Augmenting Layer system. The goal is to evolve the user's profile based on recent interactions and observed changes.

## Instructions

1. **Read the current profile** from `profiles/pro-*.md`
2. **Review recent context** — what has the user been working on? What patterns have you observed?
3. **Ask targeted questions** about changes since last assessment
4. **Update the profile** with new observations

## Update Protocol

### Quick Check (2-3 minutes)
Ask:
- "What's been your biggest work challenge since we last updated?"
- "Any new skills you've been developing or new tools you've adopted?"
- "Has your role or responsibilities shifted?"
- "Anything you've noticed about how you use AI differently now?"

### Observed Patterns
Based on your interactions, note:
- Areas where the user showed MORE expertise than their profile suggests → upgrade rating
- Areas where the user struggled MORE than expected → adjust approach
- New domains or tasks they've taken on
- Skills that seem to be developing (or atrophying)
- Changes in how critically they evaluate AI output

### Growth Tracking
Compare current vs. previous profile:
- Skills that improved: celebrate and reinforce
- Skills that plateaued: suggest targeted practice
- New goals that emerged: add to trajectory
- Red lines that need updating: ask about any changes

### Profile Update
Edit the `profiles/pro-*.md` file with:
- Updated expertise ratings (with dates of changes)
- New observations and patterns
- Adjusted task classification if needed
- Updated growth trajectory
- Any new interaction preferences discovered

### Change Log
Add an entry to the profile's change log:
```
## Change Log
- [DATE] — Updated [what changed] based on [observation/conversation]
```

## Important Notes
- Keep updates focused — don't re-do the full assessment
- Highlight growth! People like knowing they're improving
- Be honest about areas where de-skilling might be happening
- Suggest specific actions for the next period
